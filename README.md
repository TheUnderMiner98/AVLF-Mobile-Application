# Avlf Mobile

## 0. Requirements
```
node: v16.13.2
npm: 9.3.1
Angular: 15.2.1
```

## 1. Setup
Run the following commands in the desktop folder for setting up the application on your desktop
```
$ npm i
$ ng serve -o
```

## 2. Layout
